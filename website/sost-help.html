SAFETY FIRST
Read this once before anything else:
  • SOST will never ask for a private key, recovery words or wallet password.
  • Useful Compute rewards are postponed. Infrastructure is live for testing only.
  • Solo mining has high variance. Expect long gaps with no blocks.
  • SOST is in pre-market testing. There is no exchange listing and no guaranteed value.

HOW THIS PAGE WORKS

Static, reviewed, no chatbot

The answers on this page come from a human-reviewed export of the SOST AI Engine. The engine itself is internal and is not reachable from this page. Search runs locally in your browser against data/public_help_index.json; no query is sent to a server. The page works on any modern browser with JavaScript; without it you still see the static fallback above.

Every answer carries a last_reviewed date and is filtered by a public-claim guard before it is allowed to land here. If you spot wording that looks like overclaim, please report it on the BitcoinTalk thread.
