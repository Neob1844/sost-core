RESEARCH — PHASE 3 HONEST VALIDATION

Phase 2 achieved AUC 0.9995 — but that was 95% spatial memorization. Pixels from the same geological block leaked between training and test sets.

Phase 3 uses SPATIAL BLOCK CROSS-VALIDATION: the study area is divided into 10km blocks. Entire blocks are held out for testing. No pixel from a test block ever appears in training.

Result: AUC drops from 0.9995 to 0.6844.
This is NOT a failure. This IS the real signal.

WHAT 0.68 AUC MEANS:

Random guessing:      0.50 AUC
GeaSpirit Phase 3:    0.68 AUC  ← 36% better than random
Industry standard:    0.75-0.85 AUC (with proprietary data)
KoBold/Goldspot:      0.85-0.93 AUC (with $50M+ investment)

The model DOES find real geological signal. It correctly identifies 28% of deposits with 61% precision. For a system built entirely on public satellite data, this is a defensible starting point.

KEY INSIGHT:

"Methodological honesty reduced the metric but increased the real value of the system."

Phase 2 numbers were impressive but useless. Phase 3 numbers are modest but trustworthy. This is the correct foundation for a production mineral intelligence system.
