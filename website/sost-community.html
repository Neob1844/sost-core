SECTION 02

How to Participate

MINE

Run a Miner

CPU-only ConvergenceX mining. 4 GB RAM required per thread. No GPU, no ASIC. Build from source and start mining with sost-miner. Every block you mine allocates 25% to the Gold Reserve automatically.

NODE

Run a Full Node

Validate the chain independently. Every full node strengthens the network by relaying blocks and transactions, maintaining the UTXO set, and enforcing consensus rules.

DEVELOP

Contribute Code

C++17 codebase, MIT licensed. Open issues, submit pull requests, review code, write tests. All development is public on GitHub.

PoPC

Prove Gold Custody

When PoPC activates (target: February 2027), prove custody of XAUT/PAXG and earn rewards from the PoPC Pool. Model A (self-custody bonds) or Model B (timelocked escrow).
