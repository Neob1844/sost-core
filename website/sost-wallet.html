Send SOST to any address. Select UTXOs, enter destination sost1... address and amount, sign the transaction with your private key. The signature never leaves your browser. Fee auto-calculated.

Generate Wallet

Create a new secp256k1 key pair. Generates a 12-word seed phrase. Keys encrypted with AES-256-GCM before being stored in your browser. NEVER share your seed phrase or private key.

Generate → Set password → Save seed phrase securely

Import Wallet

Import an existing wallet using your 12-word seed phrase or private key in hex format. Useful for recovering access from another device or restoring from a backup.

Import → Enter seed or hex key → Set password

Backup

Export your encrypted wallet as a JSON file. Store it in a safe location. You will need your password to restore it. The backup contains your encrypted keys — without the password, it is useless.

Backup → Download encrypted JSON → Store safely

2FA (TOTP)

Add an extra security layer. Requires a TOTP code (Google Authenticator, Authy) to confirm transactions and access the dashboard. Optional but recommended.

2FA tab → Scan QR with authenticator app → Enter code

Settings

Configure: RPC node URL, auto-lock timeout (default 5 minutes), change password, visual theme. All settings stored locally in your browser.

Settings tab → Adjust preferences → Save
How it works: The password is local only — it encrypts your keys with AES-256-GCM in the browser. It is never sent to any server. Your private keys never leave your device. The wallet connects to the SOST node via RPC (getbalance, listunspent, sendrawtransaction) for blockchain data and transaction broadcast.

Proof of Personal Custody

Bond your SOST to prove custody of real gold (XAUT/PAXG)

How It Works

1. You hold XAUT or PAXG in your Ethereum wallet

2. You lock a SOST bond (12-30% of gold value) on the SOST chain

3. Random audits verify your gold balance via Ethereum RPC

4. If you maintain custody: bond returned + reward (up to 22% APR)

5. If you sell your gold: bond is slashed automatically

Your gold NEVER leaves your wallet. Only the SOST bond is at risk.
