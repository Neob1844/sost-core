Block Explorer

Explore the Chain.
Verify Everything.

The SOST block explorer is a standalone HTML application that connects directly to your node's JSON-RPC. Real-time chain data, Gold Reserve tracking, PoPC Pool monitoring, difficulty analysis, and full block/address/transaction lookup.

SECTION 01

Explorer Features

DASHBOARD

Chain Overview

Block height, total supply, hashrate estimate, current difficulty, mempool status, and network peer count. Auto-refreshes every 10 seconds.

RESERVE

Gold Reserve Tracker

Real-time Gold Vault balance, accumulated SOST, conversion status. Tracks the 25% coinbase allocation flowing to the Metals Funding Vault.

PoPC

PoPC Pool Monitor

PoPC Pool balance and accumulation tracking. 25% of every block reward flowing to the custody reward pool, visible in real time.

DIFFICULTY

Difficulty Analysis

ASERT difficulty progress bar, cASERT level indicators (L1-L5), chain timing panel, and time offset tracking.

BLOCKS

Block Detail

Full block view with header fields, coinbase transaction, cASERT mode, ConvergenceX proof data, and all transactions.

ADDRESSES

Address View

Balance with mature/immature breakdown, full UTXO list, transaction history, and coinbase maturity status.

EMISSION

Emission Curve

Interactive supply chart showing Feigenbaum exponential decay, epoch boundaries, and projected supply over time.

SEARCH

Smart Search

Search by block height, block hash, transaction ID, or address. Auto-detects input type and routes to the correct view.

SECTION 02

Connecting to Your Node

The explorer is a standalone HTML file that connects to any SOST node's JSON-RPC endpoint. No server required — it runs entirely in your browser.

Setup
3 STEPS
# 1. Start your node with RPC enabled ./sost-node --genesis genesis_block.json --chain chain.json \ --rpc-user myuser --rpc-pass mypass --profile mainnet # 2. Open explorer.html in your browser # 3. Enter RPC credentials when prompted: # Host: http://localhost:18232 # User: myuser # Pass: mypass
RPC Methods Used
JSON-RPC 2.0
MethodExplorer Use
getinfoDashboard — height, difficulty, hashrate, mempool
getblockcountChain height display
getblockhashBlock navigation by height
getblockBlock detail view — header, transactions, cASERT
getaddressinfoAddress balance, UTXO list

getmempoolinfoMempool size and fee stats
