SOST PROTOCOL

Privacy Policy

Last updated: April 1, 2026

// 1. What This App Is

SOST is a Progressive Web App (PWA) and Android application that provides access to the SOST Protocol blockchain explorer, wallet tools, and network dashboard. It connects to the public SOST mainnet via JSON-RPC.

// 2. Data We Collect

We do not collect any personal data.

  • No user accounts required for basic explorer features
  • No analytics or tracking scripts
  • No cookies for advertising or profiling
  • No third-party analytics (no Google Analytics, no Facebook Pixel)
  • No user behavior tracking
  • No device fingerprinting

// 3. Data the App Accesses

The app accesses only public blockchain data:

  • Block headers, transactions, and mempool (public blockchain data)
  • Gold prices from CoinGecko API (public market data)
  • Exchange rates from open.er-api.com (public forex data)

None of this data is personal or identifiable.

// 4. Data We Do NOT Collect

  • No names, emails, phone numbers, or addresses
  • No location data
  • No camera or microphone access
  • No contact lists
  • No browsing history
  • No advertising identifiers
  • No financial data beyond public blockchain transactions

// 5. Local Storage

The app uses browser localStorage for user preferences only:

  • Theme color selection
  • Sound on/off preference
  • Music mute preference
  • RPC endpoint URL (default: public SOST node)

This data is stored locally on your device and never transmitted to any server.

// 6. Restricted Sections

Some sections (GeaSpirit technical data, Materials Engine discovery console) require authentication. Authentication is handled server-side. No credentials are stored in the app.

// 7. Third-Party Services

The app connects to:

  • SOST RPC node (public blockchain API) — no personal data sent
  • CoinGecko API — for gold token prices — no personal data sent
  • Open Exchange Rates API — for forex rates — no personal data sent

These services have their own privacy policies. We send no personal data to any of them.

// 8. Children

This app is not directed at children under 18. We do not knowingly collect data from minors.

// 9. Changes

We may update this policy. Changes will be posted on this page with an updated date.
