# GeaSpirit Phase 53 — Remote-first public note

Public canonical score remains **25.1/40** from Phase 48.

Phase 51/53 remote-first results are **case-study and architecture validation**,
not yet a promoted replacement for the public canonical score.

## Key public result

- Pebble Alaska remote-first reached **24.2/40 (60.5%)**
- This was achieved **without requiring local geophysics**
- GeaSpirit operates from **satellite + terrain + public evidence**
- Local geology, geophysics and field datasets remain **optional bonus layers**

## 3-tier architecture

| Tier | Description | Required? |
|------|-------------|-----------|
| **Tier 1** | Satellite + terrain (S2, Landsat, DEM, NDVI) | Always available |
| **Tier 2** | Public evidence (Macrostrat, Mindat, literature) | When available |
| **Tier 3** | Local geophysics/geochemistry | Optional bonus |

## Why this matters

GeaSpirit is moving toward a genuine **remote-first global mineral search engine**
rather than depending on rare local datasets for every target area.

When local geophysics IS available, results improve dramatically
(Pebble gravity alone: +0.194 AUC). But the system works without them.
