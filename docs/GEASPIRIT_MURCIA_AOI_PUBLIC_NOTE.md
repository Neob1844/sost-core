# GeaSpirit — Murcia AOI Capability Audit

**Date:** 2026-03-30

## Summary
GeaSpirit was audited at a Murcia, Spain AOI (Campos del Río, Alguazas, Las Torres de Cotillas) to assess stack observability and Spanish public geoscience source availability.

- 7 Spanish public sources verified accessible: IGME MAGNA50 (3 geological sheets), SIGEOF geophysics portal, CNIG terrain models, IGN gravity maps, NASA EMIT (18 granules)
- 4 frontier families testable at this location
- All 5 non-testable ideas remain non-testable
