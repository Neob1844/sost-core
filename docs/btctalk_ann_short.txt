[size=14pt][b]What is SOST?[/b][/size]

SOST is a CPU-mineable cryptocurrency where every block automatically allocates 25% of the coinbase reward to a Gold Vault. This is a consensus rule — not a governance decision. Any block without the correct 50/25/25 split is rejected by every node. No premine, no ICO, no dev tax.

The Gold Vault accumulates SOST, which is periodically converted to XAUT and PAXG via a rule-based TWAP policy and deposited into a sealed Heritage Reserve Vault on Ethereum mainnet. SOST is not pegged to gold — there is no redemption right. The reserve ratio is a market-observable metric.

[size=14pt][b]ConvergenceX v2.0[/b][/size]

Not a hash lottery. Each attempt solves a 32x32 linear system via 100,000 rounds of integer-only gradient descent. Mining requires ~8GB RAM (4GB dataset + 4GB scratchpad); node validation needs only ~500MB. A unique 256-operation program per block (8 op types) prevents ASIC optimization. All consensus arithmetic is integer-only — zero floating-point in any consensus path.

[size=14pt][b]How to Mine[/b][/size]

[code]
git clone https://github.com/Neob1844/sost-core.git
cd sost-core && mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release && make -j$(nproc)
./sost-node --genesis genesis_block.json --chain chain.json --rpc-user USER --rpc-pass PASS --profile mainnet
./sost-miner --address sost1YOUR_ADDRESS --genesis genesis_block.json --chain chain.json --rpc 127.0.0.1:18232 --rpc-user USER --rpc-pass PASS --threads 4
[/code]

Requirements: Linux x86_64, 8 GB RAM per mining instance (4 GB dataset + 4 GB scratchpad); 16 GB recommended. Node-only: ~500 MB, 4 GB VPS sufficient. CPU only (GPU provides no advantage).

Chain selection: best chain by cumulative work (same as Bitcoin). Atomic fork resolution with full rollback. MAX_REORG_DEPTH = 500 blocks.
